# Melissa Magoma | Data Analytics Portfolio

I'm a Business Intelligence Analyst with 7+ years building analytics infrastructure for executive and operational audiences. My work sits at the intersection of SQL, data modeling, and AI workflow engineering, turning messy, high-volume support data into decisions that actually land.

Most recently at GitLab, I owned the end-to-end support analytics function: from production Snowflake/dbt pipelines and Tableau dashboards, to LLM-powered reporting agents built on the Anthropic Claude API with MCP tool integrations. I care about data quality as much as delivery speed, and I've learned that the two are rarely in conflict if you build things right from the start.

## What's in here

Each case study is a self-contained project folder with context, approach, and outcome. The work spans investigative analysis, infrastructure builds, and AI pipeline design, all drawn from real production environments.

| Project | What it's about |
|---|---|
| [LLM Reporting Pipeline](./llm-reporting-pipeline/) | Agentic reporting system that automates weekly executive bulletins using the Claude API, MCP integrations, and a structured prompt skill library |
| [Ticket-to-Product Attribution](./ticket-product-attribution/) | LLM-powered pipeline that classifies support tickets to product categories at scale, enabling product teams to act on customer signals |

| [SQL Samples](./sql-samples/) | Three production Snowflake queries: ticket-to-product enrichment, issue/MR tag parsing, and the WoW report base — powering CEI reporting shared with CTO, CIO, and product teams |
| [Dashboards](./dashboards/) | Production Tableau dashboard for a B2B SaaS support team — eight KPIs in a single pane of glass view, with a working universal filter built on Tableau parameters across multiple data sources |
